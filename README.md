# Task Trello API

A training REST API built with **Java + Spring Boot**, inspired by a simplified Trello-like task management system.

The goal of this project is to practice core backend concepts such as CRUD operations, DTOs, pagination, filtering, enums, global exception handling, logging and API documentation with Swagger/OpenAPI.

## Tech Stack

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Bean Validation
- Lombok
- Swagger / OpenAPI
- Maven

## Features

- Create tasks
- List tasks with pagination
- Get task by ID
- Update tasks
- Delete tasks
- Mark task as completed
- Filter tasks by status
- Filter tasks by priority
- Global exception handling
- Swagger API documentation
